 Calculator Web Application
 
📌 Project Overview:
This project is a basic Calculator Web Application developed as Task 1 of Level 2 during the internship at Oasis Infobyte.
The calculator performs simple mathematical operations such as addition, subtraction, multiplication, and division with an interactive user interface.

🚀 Features:
Perform basic arithmetic operations
User-friendly interface
Interactive buttons for input
Display screen to show input and results
Responsive design using CSS

🛠️ Technologies Used:

HTML – Structure of the calculator
CSS – Styling and layout design
JavaScript – Calculator logic and functionality

Learning Outcome:
Successfully implemented a functional calculator using basic web development technologies, improving understanding of JavaScript logic and UI design.
